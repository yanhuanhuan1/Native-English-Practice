# 英语口语化跟练

本地优先的英语口语表达练习网页应用。把中文脑子里的表达换成更本土、更像真人会说出口的英文。

## 练习流程

1. 选择难度：入门、进阶、高阶。
2. AI 生成一个真实生活对话场景。
3. 输入一句你会自然说出口的英文。
4. 提交后查看口语化反馈、更自然版本和替代表达。

输入快捷键：`Enter` 提交，`Ctrl + 空格` 换行。

## AI 服务

使用 **DeepSeek** API（`deepseek-chat` 模型）。

部署时在 Vercel 环境变量里配置：

| 变量 | 说明 |
|------|------|
| `API_KEY` | DeepSeek API Key |
| `ADMIN_PASSWORD` | 管理员密码（用于打开设置面板） |

## 本地运行

```powershell
npm.cmd install
npm.cmd run dev
```

打开 `http://127.0.0.1:3000`

或直接双击 `启动口语跟练.cmd`。
